BMB Mortgage Calculator - iPad Prototype

Launch Screen

This View consisted of an ImageView along with four button. The ImageView had a timed animation that would switch between various images. When an image was pressed, the user would be navigated to the "Mortgage Help Videos", and would begin playing the correct video. The Buttons had static data that would be passed to the "Mortagage Calculater".

Mortgage Products

I wrote a feature that would aloud the user to sort through a number of different mortgages. The way the mortgages sort, is similar to how you would sort through items on Amazon, or Zillow. I wanted to keep it easy for the user to understand.


Mortgage Products

As the user pressed a "Fixed Period" box, the user would start seeing the mortages change based on year. The user also had the option to check "Mortgage Type", that would allow to filter by year.
